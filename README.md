# 🚀 ORBITA NUMERIX
Smart Statistical Pattern Generator

Orbita Numerix adalah web generator angka berbasis analisis pola statistik sederhana dan sistem pseudo-random berbobot.

Dirancang dengan tampilan clean, ringan, dan mobile-friendly tanpa fitur ribet.


==================================================
✨ FITUR UTAMA
==================================================

- Support Mode 2D – 5D
- Input 1–5 riwayat angka terakhir
- Sistem analisis otomatis (tanpa pilih strategi)
- Generator kombinasi berbasis frekuensi & pola posisi
- 1–10 angka recommended
- Penguatan angka selanjutnya
- Tampilan ringan & HP friendly


==================================================
📘 CARA MENGGUNAKAN
==================================================

1. Pilih mode digit (2D, 3D, 4D, atau 5D).
2. Masukkan riwayat angka terakhir (maksimal 5 data).
   Pisahkan dengan koma.

   Contoh 4D:
   1234, 5678, 9012

3. Klik tombol Generate.
4. Sistem akan otomatis menghitung dan menampilkan hasil.


==================================================
📊 PENJELASAN HASIL
==================================================

1️⃣ HASIL GENERATOR UTAMA
Menampilkan 20 kombinasi angka berdasarkan:
- Pola posisi digit
- Frekuensi kemunculan digit
- Sistem pseudo-random berbobot

2️⃣ 1–10 ANGKA RECOMMENDED
Angka unik yang disaring dari hasil generator utama.

3️⃣ PENGUATAN ANGKA SELANJUTNYA
Kombinasi digit terkuat berdasarkan frekuensi tertinggi tiap posisi.


==================================================
🧠 CARA KERJA SISTEM
==================================================

Orbita Numerix menggunakan kombinasi metode:

✔ Statistik Pola Posisi
Menganalisis digit berdasarkan posisi (ribuan, ratusan, puluhan, satuan).

✔ Frekuensi Angka
Digit yang sering muncul diberi bobot lebih tinggi dalam sistem.

✔ Pseudo-Random Weighted
Pemilihan tetap acak, tetapi mengikuti distribusi bobot statistik.

==================================================
🛠 CARA AKTIFKAN DI GITHUB PAGES
==================================================

1. Upload file index.html ke repository.
2. Masuk ke Settings → Pages.
3. Pilih branch main.
4. Klik Save.
5. Website aktif dalam beberapa detik.
